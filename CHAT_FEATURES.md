# 🚀 سیستم چت پیشرفته ساینو

## ✨ قابلیت‌های جدید

### 🎨 طراحی و رابط کاربری

- **حالت تاریک/روشن**: امکان تغییر تم به صورت زنده
- **ریسپانسیو**: سازگار با تمام اندازه صفحه‌ها
- **انیمیشن‌های روان**: حرکت‌های طبیعی برای پیام‌ها

### 💬 قابلیت‌های پیام‌رسانی
- **ارسال فایل**: پشتیبانی از تصاویر، PDF، Word و...
- **Drag & Drop**: کشیدن و رها کردن فایل‌ها
- **ضبط صدا**: ضبط و ارسال پیام صوتی
- **اسکرین‌شات**: گرفتن و ارسال تصویر صفحه
- **اموجی**: انتخابگر اموجی داخلی

### 🔧 مدیریت پیام‌ها
- **ویرایش پیام**: امکان ویرایش پیام‌های ارسالی
- **حذف پیام**: حذف پیام‌های ناخواسته
- **کپی پیام**: کپی متن پیام‌ها
- **ذخیره پیام**: نشانه‌گذاری پیام‌های مهم
- **اشتراک‌گذاری**: به اشتراک گذاری پیام‌ها

### 🎭 واکنش‌ها و تعامل
- **واکنش اموجی**: افزودن واکنش به پیام‌ها
- **پیشنهادات سریع**: دکمه‌های پاسخ سریع
- **نشانگر تایپ**: نمایش وضعیت "در حال نوشتن"
- **اعلان‌های صوتی**: صدای اطلاع برای پیام‌های جدید

### 🔍 جستجو و تنظیمات
- **جستجوی زنده**: جستجو در تاریخچه پیام‌ها
- **هایلایت نتایج**: برجسته کردن کلمات جستجو شده
- **تنظیمات شخصی**: تنظیم صدا، اسکرول، زمان و...
- **Sidebar تنظیمات**: پنل جانبی برای تنظیمات

### 📱 قابلیت‌های پیشرفته
- **حالت تمام صفحه**: نمایش در حالت Fullscreen
- **صفحه‌بندی**: بارگذاری تدریجی پیام‌ها
- **شمارش کاراکتر**: نمایش تعداد کاراکترهای تایپ شده
- **اسکرول هوشمند**: اسکرول خودکار به پیام‌های جدید

### 💾 مدیریت داده‌ها
- **پاک کردن تاریخچه**: حذف کامل تاریخچه چت
- **دانلود تاریخچه**: دانلود پیام‌ها به صورت فایل متنی
- **ذخیره تنظیمات**: حفظ تنظیمات کاربر در دیتابیس
- **مدیریت فایل‌ها**: حذف خودکار فایل‌های مرتبط

## 🛠️ فایل‌های API جدید

### 📁 ساختار فایل‌ها
```
api/
├── save_settings.php    # ذخیره تنظیمات کاربر
├── save_message.php     # ذخیره/نشانه‌گذاری پیام‌ها
├── edit_message.php     # ویرایش پیام‌های کاربر
├── delete_message.php   # حذف پیام‌های کاربر
├── add_reaction.php     # افزودن واکنش به پیام‌ها
├── clear_chat.php       # پاک کردن تاریخچه چت
└── download_chat.php    # دانلود تاریخچه چت
```

### 🗃️ جداول دیتابیس جدید
```sql
-- تنظیمات کاربران
CREATE TABLE user_settings (
    id INT AUTO_INCREMENT PRIMARY KEY,
    user_id INT NOT NULL,
    setting_key VARCHAR(50) NOT NULL,
    setting_value JSON,
    UNIQUE KEY unique_setting (user_id, setting_key)
);

-- پیام‌های ذخیره شده
CREATE TABLE saved_messages (
    id INT AUTO_INCREMENT PRIMARY KEY,
    user_id INT NOT NULL,
    message_id INT NOT NULL,
    saved_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    UNIQUE KEY unique_save (user_id, message_id)
);

-- واکنش‌های پیام‌ها
CREATE TABLE message_reactions (
    id INT AUTO_INCREMENT PRIMARY KEY,
    user_id INT NOT NULL,
    message_id INT NOT NULL,
    emoji VARCHAR(10) NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    UNIQUE KEY unique_reaction (user_id, message_id, emoji)
);
```

## 🎯 ویژگی‌های کلیدی

### 🚀 عملکرد بهینه
- **بارگذاری تدریجی**: صفحه‌بندی پیام‌ها برای عملکرد بهتر

- **Validation**: اعتبارسنجی کامل ورودی‌ها
- **File Security**: بررسی نوع و اندازه فایل‌ها

### 📱 تجربه کاربری
- **واکنش‌پذیری**: پاسخ فوری به تعامل کاربر
- **راهنمایی**: tooltip ها و راهنمای بصری

- **دسترسی**: سازگار با کلیدهای میانبر
